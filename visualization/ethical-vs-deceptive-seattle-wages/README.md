# Visual Trust and Misleading Analytics

Visual analytics case study comparing trustworthy and intentionally misleading representations of City of Seattle wage data.

## Portfolio Files

- `reflection.pdf`: design rationale and ethics reflection.
- `ethical-visualization.pdf`: visualization using average hourly rate by department with clear labels and proportional encoding.
- `deceptive-visualization.pdf`: intentionally misleading visualization using maximum hourly rate and distorted axis choices.
- `specs/ethical_average_hourly_rate.vl.json`: cleaned Vega-Lite spec for the trustworthy chart.
- `specs/deceptive_max_hourly_rate.vl.json`: cleaned Vega-Lite spec for the intentionally misleading chart.

## Skills Demonstrated

- Data ethics and visual rhetoric.
- Chart specification and visual encoding with Vega-Lite.
- Public-sector wage data exploration.
- Aggregation choice and axis-scale reasoning.
- Written explanation of visualization tradeoffs.

## Portfolio Boundary

The public repo includes the chart specifications and design rationale, while private source materials remain excluded. The misleading chart is included only as a critique artifact to show how aggregation and scale choices can distort interpretation.
