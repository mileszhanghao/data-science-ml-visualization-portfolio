# Vega-Lite Specs

These specifications rebuild the paired wage visualizations from public City of Seattle compensation data.

Run them in any Vega-Lite renderer, Observable notebook, or frontend dashboard that supports Vega-Lite JSON.

Data source: `https://data.seattle.gov/resource/2khk-5ukd.csv`

The ethical chart uses mean hourly rate and a zero-based quantitative axis. The deceptive chart intentionally switches to maximum hourly rate and a truncated axis to demonstrate how plausible-looking encodings can overstate differences.
