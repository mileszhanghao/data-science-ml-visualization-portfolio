# Data Science, ML, and Visualization Portfolio

Selected data science coursework and projects cleaned for public portfolio review.

This repository emphasizes model building, validation, visual communication, and data ethics. Raw course prompts, grading artifacts, instructor materials, restricted notebooks, and private datasets are not included.

## Projects

### Ethical vs. Deceptive Visualization

`visualization/ethical-vs-deceptive-seattle-wages/`

Built two Vega-Lite/Observable visualizations from City of Seattle wage data to compare ethical and intentionally deceptive design choices. The reflection explains how aggregation, axis scaling, labels, and context influence interpretation.

### Minimum Wage vs. Cost of Living

`visualization/minimum-wage-cost-of-living/`

Group data visualization project comparing minimum wage and cost-of-living indicators across U.S. regions using BLS/CPI-related data sources. The public copy is labeled as a group project and preserves the report without grading material.

### Course Certification Modeling

`machine-learning/course-certification-modeling/`

Machine learning workflow for predicting course certification from learner activity features. The public summary documents the modeling approach without publishing the restricted class notebook.

### CIFAR-10 Deep Learning Experiments

`machine-learning/cifar10-deep-learning/`

PyTorch image-classification experiments comparing linear, multilayer, convolutional, and regularized CNN architectures. The public summary documents model design and evaluation concepts without publishing the restricted class notebook.
