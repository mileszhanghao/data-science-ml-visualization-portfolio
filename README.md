# Data Science, ML, and Visualization Portfolio

Data science portfolio organized around model development, visual analytics, and decision-support communication. The projects show how I move from raw behavioral, image, wage, and economic data into validated models or stakeholder-ready visual narratives.

## Projects

### Visual Trust and Misleading Analytics

`visualization/ethical-vs-deceptive-seattle-wages/`

Built paired visualizations from City of Seattle wage data to show how aggregation, axis scaling, labels, and missing context can change decision-making. The public version includes cleaned Vega-Lite specifications so the design choices are visible as code, not only as PDFs.

### Minimum Wage vs. Cost of Living

`visualization/minimum-wage-cost-of-living/`

Collaborative data story comparing minimum wage and cost-of-living indicators across U.S. regions using public economic data. The project is presented as a policy and affordability dashboard/report case study.
The portfolio writeup now documents the dashboard architecture, linked views, and data-merge model separately from the submitted report.

### Course Certification Modeling

`machine-learning/course-certification-modeling/`

Machine learning workflow for predicting learner certification from behavioral activity features. The public summary documents feature engineering, class imbalance handling, model comparison, and ROC/AUC evaluation.

### CIFAR-10 Deep Learning Experiments

`machine-learning/cifar10-deep-learning/`

PyTorch image-classification experiments comparing linear, multilayer, convolutional, and regularized CNN architectures. The public summary documents model design and evaluation tradeoffs.
